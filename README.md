# Fraud Detection System 

 ## About
This project simulates a simple fraud detection mechanism for bank transactions.  
It automatically blocks accounts that perform more than **3 transactions within 2 minutes**.

---

 ## Technologies Used
- Java 21
- Jakarta Servlet API
- Apache Tomcat 
- MySQL 
- Log4j2
- JDBC
- JUnit
- JaCoCo
- SonarQube
- Maven
- Postman
- Mockito

---

 ## Features
- Java Servlet backend
- MySQL database
- DAO-based data layer
- Fraud detection logic
- REST-style POST endpoint `/Transaction`
- Automatic account blocking and fraud alert logging
- Log4j-based logging

---

## Testing
- JUnit – Used to write and run unit test cases.
- Mockito – Used to mock dependencies and isolate components during testing.
- JaCoCo – Used to measure and generate code coverage reports.
- SonarQube – Used to analyze code quality, coverage, and detect issues.
